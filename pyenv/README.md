## Install

**Install [Pyenv](https://github.com/pyenv/pyenv)**

List available installs

```bash
pyenv install -l
```

Select the latest, stable version

```bash
pyenv install 3.11.1
```

Set as global

```bash
pyenv global 3.11.1
```
