# MacOS Dotfiles

## Install

**Install [Rosetta](https://support.apple.com/en-us/HT211861)**

```bash
bash macos/install.sh
```

## Usage

### Finder

1. Open a new Finder window
2. Press **CMD + ,**

**General**

_Show these items on the desktop_

- ☒ Hard disks
- ☒ External disks
- ☒ CDs, DVDs, and iPods
- ☑ Connected servers

_New Finder windows show_

- Select your user directory

**Tags**

Disable all tags.

**Sidebar**

_Favorites_

- ☒ Recents
- ☑ Applications
- ☒ Desktop
- ☒ Documents
- ☑ Downloads
- ☒ Movies
- ☒ Music
- ☒ Pictures
- ☑ [User Directory]

_Locations_

Uncheck all.

_Tags_

Uncheck all.

**Advanced**

- ☑ Show all filename extensions
- ☒ Show warning before changing extension
- ☑ Show warning before removing from iCloud Drive
- ☑ Show warning before emptying Trash
- ☑ Remove items from trash after 30 days

_Keep folders on top_

- ☒ In windows when sorting by name
- ☒ On Desktop

_When performing a search_

Select **Search the Current Folder**

## System Settings

1. Open Spotlight → **CMD + Space**
2. Enter **System**
3. Select **System Preferences.app**

### General

- Appearance → **Auto**
- Accent color → **Multicolor**
- Highlight color → **Accent Color**
- Sidebar icon size → **Medium**
- ☑ Allow wallpaper tinting in windows
- Show scroll bars → **Automatically based on mouse or trackpad**
- Click in the scrollbar to → **Jump to the next page**
- Default web browser → **Google Chrome.app**
- Prefer tabs → **Always**
- ☒ Ask to keep changes when closing documents
- ☑ Close windows when quitting an app
- Recent items → None
- ☒ Allow Handoff between this Mac and your iCloud devices

### Desktop & Screensaver

Select desired Desktop & Screensaver.

### Dock & Menu Bar

_Dock_

- Size slider → **33%**
- ☑ Magnification
- Magnification slider → **50%**
- Position on screen → **Bottom**

> Remaining defaults are fine

_Menu Bar_

- ☒ Automatically hide and show the menu bar on desktop
- ☒ Automatically hide and show the menu bar in fullscreen

_Control Center_

- Uncheck **Show in Menu Bar** for all items.
- Check **Show in Control Center** for
  - Battery

_Control Center: Clock_

- Data Options: Show data → **when space allows**
- ☑ Show day of the week
- Time Options → Digital
- ☒ Use a 24-hour clock
- ☒ Show AM/PM
- ☒ Flash the time separators
- ☒ Display the time with seconds
- ☒ Announce the time

### Mission Control

- ☒ Automatically rearrange Spaces based on most recent use
- ☑ When switchign to an application, switch to a Space with open windows for the application
- ☒ Group windows by application
- ☑ Displays have separate spaces

> Remaining defaults are fine
